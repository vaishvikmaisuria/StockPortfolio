## Setup 

### run the react frontend locally
> npm start

### run the go backend locally
> go mod init github.com/vaishvikmaisuria/Willow

> go mod tidy

> go mod download

> go mod verify

> go run main.go

### run the frontend and backend with docker
#### on Linux 
> make build
#### on Windows
> docker-compose up --build
